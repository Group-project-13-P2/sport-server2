##    Register a new user. 

* **URL**

    /register

* **Method:**

    `POST`

*  **Data Params**

   **Required:**
 
   `email=[string]`
   `password=[string]`

* **Success Response:**

  * **Code:** 201 <br />
    **Content:** 
    ```md
    {
        "id": 1,
        "email": "bulba@gmail.com"
    }
    ```
* **Error Response:**

  * **Code:** 500 <br />
    **Content:** `{ error : "Internal Server Error" }`
